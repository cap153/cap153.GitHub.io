pacman的使用

2021-07-18

sudo pacman
[ -Syyu 强行更新软件]
-Ss+ search，查询一个软件
-Sc 删除缓存，pacman更新软件后默认不会删除安装包
-R + 删除某软件
-Rs+ 删除某软件及其依赖
[ -Rns+ 连同全局配置一起删除]
-Q 显示所有已安装软件 |wc -l 显示有几行
-Qe 显示自己安装的软件
-Qs+ 查询已安装的某软件
-Qdt 查询不再需要的软件
-Qdtq 查询不再需要的软件，且不显示版本号
[ -R (pacman -Qdtq)]
删除不再需要的软件
