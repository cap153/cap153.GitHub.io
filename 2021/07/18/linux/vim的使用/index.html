vim的使用

2021-07-18

jkli 光标移动

JKLI 快速光标移动
ctrl j 移到行首
ctrl l 移到行尾
ctrl i 移到文档首部，同gg
ctrl k 移到文档尾部，同G
n 插入
N 行首插入
A 尾部插入
u 撤销
ctrl r 取消撤销
S 保存
Q 退出
R 重新加载配置文件
c 删除并插入，即change
cw 改变下一个单词
cb 改变上一个单词
cnw 改变当前光标所在位置的单词，change new word
cn” 改变“”中的所有内容
d 删除
y 复制
p 粘贴
w 下一个单词,word
b 上一个单词,back
zz make this line in center
gg 将光标移动到行首
G 将光标移动到最后一行
0 将光标移动到本行的开头
$ 将光标移动到本行的末尾
find，接查找的字母,将会从光标当前位置向后查找并跳转位置向后查找并跳转
/ 搜索 - 上一个 = 下一个
空格加回车 退出搜索高亮
ctrl a 增加数字

v 可视模式
V 可视行模式
ctrl v 可视块模式

sj 向左边分屏
sl 向右边分屏
si 向上分屏
sk 向下分屏
空格 jkli 分屏后上下左右移动
方向键 更改分屏大小
fj 新建标签页
fn 向左移动标签页

:r ！ls 插入当前目录下的所有文件名
:e+ 打开文件，edit编辑
空格gy 沉浸模式
m+ 打标签
ctrl+o 回到上一次编辑的位置，可以跨文件
ctrl+i 回到之前编辑的位置，反复横跳
gf 进入文件，go to file
:w !sudo tee % 管理员权限保存
:%TOhtml 生成html文件
:tt 打开文件目录树
:tr 翻译当前单词
“ayy 把当前行的内容复制到a这个键
:term 打开一个终端

ysks ‘ surround with ‘
