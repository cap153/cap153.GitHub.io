git基本操作

初始化操作

指令 含义
git init 把当前文件夹变成git仓库
git config –global user.name “captain” 设置提交的用户名
git config –global user.email “” 设置提交者的邮箱
git config –global core.editor nvim 更改提交使用的编辑器
git status 查看当前文件的情况

添加修改

指令 含义
git add 文件名 添加修改
git diff 显示未被添加的更改
git reset 撤回看添加的修改

提交更改

指令 含义
git commit -m “描述” 提交更改
git rm –cached 文件名 删除对应文件的追踪

分支管理

指令 含义
git branch 分支名 创建新的分支
git branch -d 分支名 删除对应分支,D大写强制删除
git checkout 分支名 切换分支
git merge 分支名 合并分支

远程仓库

指令 含义
git remote add origin 项目的地址 把当前git仓库添加到对应地址
git push –set-upstream origin master 把主分支上传到网上
git config credential.helper store 记住github用户名和密码
git clone 仓库的链接 复制对应的仓库
git clone –depth 1 https://github.com/dogescript/xxxxxxx.git 只下载远程仓库中的最新的一个版本
git pull 把网上的更新下载到本地

忽略提交的文件
添加对应文件的文件名到.gitignore
